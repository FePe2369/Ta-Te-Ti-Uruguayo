# 🇺🇾 Ta-Te-Ti Uruguayo

Este es el clásico juego del Ta-Te-Ti con temática uruguaya.

## Descripción

Un Ta-Te-Ti modernizado con los colores de la bandera uruguaya. Utiliza el azul profundo (#001489) característico de nuestra bandera nacional, el dorado del Sol de Mayo, y una estética que refleja la identidad oriental.

## Características

- **Colores patrios**: Azul (#001489) y dorado (#FFD700)
- **Interfaz moderna**: Con animaciones suaves y diseño responsive
- **Fondo con la bandera**: Las franjas celeste y blanco de fondo, con el Sol de Mayo animado
- **Interfaz en español**
- **Responsive**: Funciona en cualquier dispositivo

## Reglas del Juego

1. El juego comienza con el jugador **X** (azul uruguayo)
2. Los jugadores se turnan para marcar las casillas
3. **X** juega en azul, **O** juega en dorado
4. El primer jugador que logre tres marcas en línea (horizontal, vertical o diagonal) gana
5. Si todas las casillas se llenan sin un ganador, la partida termina en empate
