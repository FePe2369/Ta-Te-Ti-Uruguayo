# 🇺🇾 Ta-Te-Ti Uruguayo

El clásico Ta-Te-Ti con temática uruguaya: los colores de la bandera, el Sol de Mayo animado y la interfaz en español.

## Características

- Paleta patria: azul `#001489` y dorado `#FFD700`
- Sol de Mayo animado y franjas de la bandera de fondo
- Animaciones suaves y diseño responsive
- Interfaz en español

## Cómo jugar

Abrí `index.html` en el navegador.

- Empieza **X** (azul); **O** juega en dorado
- Los jugadores se turnan para marcar casillas
- Gana quien alinee tres marcas (horizontal, vertical o diagonal)
- Si se llenan todas las casillas sin línea, es empate

## Tecnologías

HTML5 para la estructura, CSS3 para el diseño y las animaciones (gradientes, Sol de Mayo, media queries) y JavaScript puro para la lógica del juego y la detección del ganador.
