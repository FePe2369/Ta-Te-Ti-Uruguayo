# 🇺🇾 Ta-Te-Ti Uruguayo

¡Bo! Este es el clásico juego del Ta-Te-Ti, pero con toda la garra charrúa que nos caracteriza. Nada de "Tic Tac Toe" acá, esto es 100% uruguayo.

## 🎮 ¿De qué va la cosa?

Un Ta-Te-Ti modernizado con los colores de nuestra bandera. El azul profundo (#001489) que nos identifica, el dorado del Sol de Mayo, y toda la onda oriental que llevamos en la sangre.

## ✨ Características

- **Colores patrios**: Azul (#001489) y dorado (#FFD700), los de la celeste
- **Interfaz moderna**: Con animaciones suaves y diseño responsive
- **Fondo con la bandera**: Las franjas celeste y blanco de fondo, con el solcito girando
- **Todo en español**: Porque somos uruguayos, ¿qué más querés?
- **Responsive**: Funciona en cualquier dispositivo, desde el celu hasta la compu

## 🚀 ¿Cómo lo uso?

Es más fácil que patear un penal:

1. Clona el repositorio o descarga los archivos
2. Abrí el `index.html` en tu navegador
3. ¡Dale que va! Empezá a jugar

```bash
# Si tenés git instalado
git clone [tu-repo-aca]
cd tateti-uruguayo

# Abrí el index.html con tu navegador favorito
```

## 🎯 Cómo Jugar

1. El juego arranca con el jugador **X** (azul uruguayo)
2. Los jugadores se turnan para marcar las casillas
3. **X** juega en azul, **O** juega en dorado
4. El primero que hace tres en línea (horizontal, vertical o diagonal) ¡gana!
5. Si se llena el tablero sin ganador, es empate y te tomás un mate para pensar mejor la próxima

## 💡 Tecnologías Usadas

- **HTML5**: Para la estructura
- **CSS3**: Para todo el diseño y las animaciones con garra
- **JavaScript (Vanilla)**: Lógica del juego, sin frameworks ni vueltas
- **Garra Charrúa**: El ingrediente secreto 💪

## 🔥 Características Técnicas

### JavaScript

- Event listeners modernos (nada de onclick en el HTML)
- Manejo de estado del juego
- Detección inteligente de ganadores
- Animaciones coordinadas con CSS

### CSS

- Gradientes con los colores de la bandera
- Animaciones suaves (fadeIn, pulse, rotate)
- Diseño responsive con media queries
- Efectos hover modernos
- Sol animado que rota (¡como debe ser!)

### HTML

- Estructura semántica
- Data attributes para mejor manejo del DOM
- Accesible y limpio
